# Handover Checklist

Complete all items before official handover.

## Infrastructure

- [ ] Terraform state accessible
- [ ] All ECS tasks healthy
- [ ] ALB targets healthy
- [ ] CloudFront distribution active
- [ ] WAF in BLOCK mode (or approved COUNT)

## Testing

- [ ] All unit tests passing
- [ ] Integration tests passing
- [ ] Smoke tests green
- [ ] Load test completed
- [ ] Canary deployment successful

## Security

- [ ] Security sign-off obtained
- [ ] Secrets rotated post-handover
- [ ] Access controls verified
- [ ] Audit logging enabled

## Cost

- [ ] Monthly cost within budget
- [ ] Budget alerts configured
- [ ] Cost allocation tags applied

## Documentation

- [ ] Handover README reviewed
- [ ] Runbooks complete
- [ ] Architecture diagram accurate
- [ ] FAQ updated

## Training

- [ ] Onboarding session completed
- [ ] Exercises completed by team
- [ ] Q&A session held

## Operations

- [ ] On-call rotation assigned
- [ ] Escalation contacts updated
- [ ] Dashboards accessible
- [ ] Alerts configured

## Compliance

- [ ] Legal review complete
- [ ] Privacy policy updated
- [ ] Data retention policy defined
